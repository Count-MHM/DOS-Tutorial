Dos Tutorial

Installation

We are going to use Open Watcom V2.0 as our compiler, Visual Studio Code as our code editor, and DosBox as our DOS emulator.

Visual Studio Code

Let's start by installing the easier one first.

It's as simple as downloading it from Visual Studio Code's Website and installing it.

You can also install it via winget by running this command:

winget install -e --id Microsoft.VisualStudioCode

We also need to install a few Extensions to make life easier for us.

First, let's open the Extensions menu. There are few ways to do that:

  1. Look for this icon On the left side and click it!
  2. Open Command Palette (with Ctrl+Shift+P or F1) and type View: Show Extensions
  3. Use the default keyboard shortcut : Ctrl+Shift+X

From there search and install these extensions:

  1. C/C++
  2. Makefile Tools

Open Watcom V2.0

Open Watcom V2.0 is only available for Windows and Linux (dos, win16, and os2 are also supported).

I'm only going to cover Windows and Linux, macOS users may have to use Wine but I can't test it because I don't have any machine running macOS.

First we have to download Watcom so go to GitHub from there go to Releases then select the latest build (2023-07-02-Build at the time of writing)

Windows

Download open-watcom-2_0-c-win-x64.exe and run it.

Read the EULA and press I Agree.

Select a folder to install Watcom (I use C:\Dev\Programs\WATCOM but you can use any folder)

Select Full installation and hit next

Press next

If you have installed it before, you will be prompted about what to do about some DLLs

Select Install the new file and keep existing file and select Use the current choice for all subsequent DLLs

Select Modify local machine environment variables and hit OK

Select Don't Modify associations and press OK

There is one more step that I'm going to explain later. (follow DOSBox installation)

Linux

Download open-watcom-2_0-c-linux-x64.

Extract it and copy it to /usr/bin/watcom

And add these lines to the top of ~/.profile and ~/.bashrc

export WATCOM=/usr/bin/watcom
export PATH=$WATCOM/binl64:$WATCOM/binl:$PATH

Restart the terminal and run wpp in terminal and if you see Open Watcom C++ x86 16-bit Optimizing Compiler ... then you have installed it successfully!

DOSBox

Windows

Run this command in cmd

winget install dosbox

After installing it, open windows search and search for Edit the system environment variables and open it.

On the button of the opened window, there will be a button called Environment Variables... press it!

You'll see two lists we're going to change, the top one called User Variables for $(YOUR_USER_NAME)

Find the variable called path and double click it.

Press the new button and type the installation location of DOSBox. (default is C:\Program Files (x86)\DOSBox-0.74-3)

Now open a new cmd and run dosbox and if it starts DOSBox then you have installed it successfully!

Press the new button again and type the installation location of Watcom and add \binnt64. (so it'll be C:\Dev\Programs\WATCOM\binnt64 for me)

Go back to the page that you found path in.

And make these variables: (instead of ###WATCOM### write your Watcom install location without a \ at the end)

  • WATCOM = ###WATCOM###
  • EDPATH = %WATCOM%\eddat
  • INCLUDE = %WATCOM%\H;%WATCOM%\H\NT;%WATCOM%\H\NT\DIRECTX;%WATCOM%\H\NT\DDK;
  • WHTMLHELP = %WATCOM%\BINNT\HELP
  • WIPFC = %WATCOM%\WIPFC

Then press okay until you close all the windows (don't close with X)

Now open cmd and run wpp and if you see Open Watcom C++ x86 16-bit Optimizing Compiler ... then you have installed Watcom successfully!

Linux

Run this command and that's it!

sudo apt-get install dosbox
